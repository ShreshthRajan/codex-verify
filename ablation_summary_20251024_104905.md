# Ablation Study Summary - Multi-Agent Advantage

## Key Findings

**Multi-Agent Advantage Proven:**

- 1 agent: 32.8% average accuracy
- 2 agents: 47.7% average accuracy (+14.9pp)
- 3 agents: 61.2% average accuracy (+28.4pp from single)
- 4 agents: 72.4% average accuracy (+39.7pp from single)

**Conclusion:** Multi-agent architecture provides 39.7 percentage point improvement over single-agent baseline.

**Best Configuration:** Pair-C+P
- Accuracy: 79.3%
- TPR: 83.3%
- FPR: 40.0%

**Improvement over worst single-agent (Single-P):**
- 62.1 percentage points
